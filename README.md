# 🚀 Task 12 – Modern UI Components

## 📌 Overview
This project is part of the Web Development training (Task 12).  
It showcases a collection of modern and responsive UI components built using **HTML, CSS, and JavaScript**.

The goal of this project is to practice building real-world UI elements with clean design and smooth interactions.

---

## 🎨 Features

### 🧭 Navigation Bars (Header Section)
- Pill Navigation Bar with active indicator
- Expandable Sidebar Navigation
- Glass Floating Navigation Bar

### 🎴 UI Cards (Body Section)
- Gradient Border Cards
- Hover Lift & Glow Effects
- Reveal Panel Cards

### 🔗 Social Media Icons (Footer Section)
- Diagonal Wipe Hover Icons
- Floating Rise Icons with labels

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
